{% trans "1. 정의" %}

  • {% blocktrans %}"이용자"란 본 약관에 동의하고 서비스를 이용하는 개인 또는 법인을 의미합니다.{% endblocktrans %}
  • {% blocktrans %}"콘텐츠"란 이용자가 업로드하는 원본 파일(PDF/이미지 등)과 서비스가 생성·가공한 결과물을 의미합니다.{% endblocktrans %}
  • {% blocktrans %}"Eco/Precision"은 번역 처리 방식의 모델 유형으로, Precision은 높은 정밀도를 제공하나 더 많은 과금 단위를 소모할 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}"서비스 크레딧"은 금전적 환불이 아닌 향후 이용요금에 상당하는 신용으로, 환급·현금화되지 않습니다.{% endblocktrans %}

{% trans "2. 자격 및 계정" %}

  • {% blocktrans %}원칙적으로 만 14세 미만은 이용 대상이 아닙니다. 필요한 경우 법정대리인 동의가 요구될 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}이용자는 계정 접근 정보의 보안을 유지해야 하며, 계정의 무단 사용에 대해 즉시 통지해야 합니다.{% endblocktrans %}
  • {% blocktrans %}회사(서비스)는 약관 위반, 불법·유해한 사용, 타인의 권리 침해 우려가 있는 경우 계정을 제한·정지할 수 있습니다.{% endblocktrans %}

{% trans "3. 요금·결제·세금" %}

  • {% blocktrans %}요금제, 한도, 과금 규칙은 서비스 내 "Pricing" 페이지의 고지에 따릅니다.{% endblocktrans %} {% trans "자세히 보기" %}
  • {% blocktrans %}대한민국 내 소비에는 부가가치세 10%가 부과되며, 그 외의 국가에서의 이용은 관련 법령상 요건 충족 시 영세율(0%)이 적용될 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}대한민국 결제는 토스페이먼츠, 그 외의 국가의 결제는 PayPal을 사용하며, 각 결제사는 자체 약관·정책을 적용합니다.{% endblocktrans %}

{% trans "4. 크레딧" %}

  • {% blocktrans %}크레딧은 Eco 기준 1페이지=1크레딧, Precision은 1페이지=6크레딧이 차감됩니다.{% endblocktrans %}
  • {% blocktrans %}$5(미화)로 1,000 크레딧 단위 구매가 가능하며, 유효기간은 구매일로부터 1년입니다.{% endblocktrans %}
  • {% blocktrans %}월간 페이지 한도 미초과 시 크레딧은 사용되지 않으며, 한도 초과 시 또는 Precision 사용 한도 초과분에 대해 사용됩니다.{% endblocktrans %}
  • {% blocktrans %}크레딧은 양도·환불·현금화가 불가합니다.{% endblocktrans %}

{% trans "5. 자동갱신·해지·환불" %}

  • {% blocktrans %}구독은 월 단위 자동갱신되며, 갱신 24시간 전까지 계정 설정에서 해지할 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}디지털 서비스 특성상 환불은 원칙적으로 불가합니다. 시스템 장애로 결과물이 생성되지 않은 경우 서비스 크레딧을 제공할 수 있으며, 이는 환불이 아니고 월 요금의 50%를 상한으로 합니다.{% endblocktrans %}

{% trans "6. 콘텐츠 라이선스 및 지식재산" %}

  • {% blocktrans %}이용자는 업로드한 콘텐츠에 대한 권리를 보유하며, 서비스 제공·운영·보안·백업을 위해 필요한 범위 내에서 비독점적·전세계적·로열티 없는 호스팅·저장·처리 라이선스를 회사에 부여합니다.{% endblocktrans %}
  • {% blocktrans %}이용자는 업로드 콘텐츠에 대한 적법한 권리(사용·배포·번역 권한 포함)를 보유해야 하며, 제3자의 권리를 침해하지 않아야 합니다.{% endblocktrans %}
  • {% blocktrans %}서비스와 관련된 상표·로고·소프트웨어·문서 등은 회사 또는 라이선서의 지식재산이며, 약관에서 명시적으로 허용한 범위를 넘어 사용할 수 없습니다.{% endblocktrans %}
  • {% blocktrans %}이용자가 제공하는 피드백·제안은 회사가 제한 없이 사용할 수 있으며, 이용자에게 별도의 보상이 제공되지 않습니다.{% endblocktrans %}

{% trans "7. 금지행위(사용정책)" %}

  • {% blocktrans %}저작권 침해 자료, 불법물, 악성코드, 성착취물 등 불법·유해 콘텐츠의 업로드·처리가 금지됩니다.{% endblocktrans %}
  • {% blocktrans %}서비스의 리버스 엔지니어링, 우회, 과도한 트래픽 유발, 자동화된 대량 요청 등 서비스 안정성을 해치는 행위는 금지됩니다.{% endblocktrans %}
  • {% blocktrans %}타인의 권리를 침해하거나 법령을 위반하는 방식으로 결과물을 배포·상업화하는 행위는 금지됩니다.{% endblocktrans %}
  • {% blocktrans %}회사는 위반이 확인되거나 합리적으로 의심되는 경우 콘텐츠 삭제, 처리 중단, 계정 정지·해지 등 필요한 조치를 취할 수 있습니다.{% endblocktrans %}

{% trans "8. 저작권 침해 신고" %}

{% blocktrans %}권리자가 침해를 주장하는 경우, 신속한 처리를 위해 소유 증빙, 침해 주장 내역, 연락처를 포함하여 신고해 주시기 바랍니다. 회사는 관련 법령에 따라 적절한 조치를 취하고 필요한 경우 당사자에게 통지합니다.{% endblocktrans %}

{% trans "9. AI 결과물 고지·보증부인" %}

{% blocktrans %}생성형 AI의 특성상 결과물에는 오류·누락·부정확성이 포함될 수 있습니다. 이용자는 결과물을 최종 사용 전에 검수해야 하며, 결과물의 사용(배포·상업적 이용 포함)으로 인한 책임은 이용자에게 있습니다. 서비스는 “있는 그대로(AS IS)” 및 “이용 가능한 범위(AS AVAILABLE)”로 제공되며, 특정 목적 적합성, 비침해성 등에 대한 명시적·묵시적 보증을 하지 않습니다.{% endblocktrans %}

{% trans "10. 책임 제한" %}

{% blocktrans %}법이 허용하는 최대 한도에서, 회사는 간접·부수·특별·결과적 손해 또는 영업이익·데이터 손실에 대해 책임지지 않습니다. 또한, 회사의 총 누적 책임은 이용자가 문제 발생 12개월 이내에 서비스에 지불한 금액을 상한으로 합니다. 일부 지역에서는 책임 제한이 허용되지 않을 수 있으며, 이 경우 적용 법령이 허용하는 한도 내에서만 제한됩니다.{% endblocktrans %}

{% trans "11. 면책" %}

{% blocktrans %}이용자는 이용자의 콘텐츠 또는 약관·법령 위반으로 인해 발생하는 제3자의 청구·손실·책임·비용(합리적 변호사 비용 포함)으로부터 회사 및 그 임직원·파트너를 방어하고 면책해야 합니다.{% endblocktrans %}

{% trans "12. 서비스 가용성(SLA)" %}

{% blocktrans %}Starter/Growth 플랜에는 공식 SLA가 제공되지 않습니다. Pro/Enterprise 플랜은 월 99.5% 업타임을 목표로 하며, 예정된 점검·불가항력·고객사 원인 장애를 제외하고 미달 시 서비스 크레딧을 제공합니다. 서비스 크레딧은 환불이 아니며, 월 요금의 50%를 상한으로 합니다.{% endblocktrans %}

{% trans "14. 제3자 서비스" %}

{% blocktrans %}서비스는 호스팅·스토리지·결제·전송·분석 등 제3자 서비스를 이용할 수 있으며, 해당 서비스는 각 제공자의 약관·정책이 적용됩니다. 회사는 합리적인 범위에서 제3자 서비스의 안정적 연계를 위해 노력합니다.{% endblocktrans %}

{% trans "15. 서비스 변경·베타" %}

{% blocktrans %}회사는 서비스의 기능·요금·정책을 변경하거나 일부 기능을 중단할 수 있습니다. 베타·시험 기능은 “있는 그대로” 제공되며, 예고 없이 변경·중단될 수 있습니다.{% endblocktrans %}

{% trans "16. 중지·해지" %}

{% blocktrans %}이용자가 약관 또는 법령을 위반하거나, 타인의 권리를 침해하는 경우, 회사는 사전 통지 없이 서비스 제공을 전부 또는 일부 중지하거나 계정을 해지할 수 있습니다. 해지 후에도 성격상 존속되어야 하는 조항(지식재산, 보증부인, 책임 제한, 면책, 준거법 등)은 유효합니다.{% endblocktrans %}

{% trans "17. 준거법·관할" %}

{% blocktrans %}본 약관은 대한민국 법률에 따라 해석·적용되며, 본 약관 또는 서비스와 관련하여 발생하는 분쟁은 대한민국 서울중앙지방법원을 전속적 관할 법원으로 합니다.{% endblocktrans %}

{% trans "18. 통지·변경" %}

  • {% blocktrans %}회사는 서비스 내 공지, 이메일, 앱 내 알림 등 합리적인 방법으로 중요한 통지를 수행할 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}약관이 변경되는 경우, 효력 발생일 및 주요 변경 사항을 사전에 고지하며, 효력 발생일 이후 서비스 이용은 변경된 약관에 대한 동의로 간주됩니다.{% endblocktrans %}

  • {% blocktrans %}분리가능성: 어떤 조항이 무효가 되더라도 나머지 조항은 계속 유효합니다.{% endblocktrans %}

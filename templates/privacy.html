{% trans "1. 적용 범위" %}

{% blocktrans %}본 방침은 웹사이트, 모바일 웹, 그리고 서비스에 부수되는 모든 온라인 기능에 적용됩니다. 제3자 웹사이트로 연결되는 링크는 당사가 통제하지 않으므로 해당 사이트의 정책을 확인하시기 바랍니다.{% endblocktrans %}

{% trans "2. 수집하는 정보" %}

  • {% blocktrans %}계정 정보: 이메일, 사용자명, 인증 관련 식별자, 구독/결제 상태.{% endblocktrans %}
  • {% blocktrans %}서비스 이용 정보: 접속 로그, 기기/브라우저 정보, 언어설정, 오류 리포트.{% endblocktrans %}
  • {% blocktrans %}업로드 파일: 이용자가 번역을 위해 업로드하는 PDF/이미지 등 원본 데이터(콘텐츠 자체 포함).{% endblocktrans %}
  • {% blocktrans %}결제 관련 정보: 카드/계정 정보는 결제 대행사를 통해 처리되며, 당사는 필요한 거래 식별자와 결과만 보관합니다.{% endblocktrans %}
  • {% blocktrans %}고객지원 정보: 문의 내용, 첨부 파일, 응대 기록.{% endblocktrans %}

{% trans "3. 정보의 이용 목적" %}

  • {% blocktrans %}서비스 제공: 번역 처리, 파일 보관, 계정 관리, 과금 및 영수증 발행.{% endblocktrans %}
  • {% blocktrans %}개선 및 연구: 품질 향상, 오류 분석, 성능/사용성 개선(개인 식별이 필요하지 않은 범위에서 통계·집계 형태로 수행).{% endblocktrans %}
  • {% blocktrans %}보안: 계정/서비스 보호, 부정 사용 탐지 및 대응.{% endblocktrans %}
  • {% blocktrans %}법적 준수: 세무·회계, 법적 요청 대응.{% endblocktrans %}
  • {% blocktrans %}고객지원: 문의 대응 및 공지 발송(필수 안내 포함).{% endblocktrans %}

{% blocktrans %}업로드 데이터와 번역 결과물은 별도 동의가 없는 한, 모델 학습에 사용되지 않습니다.{% endblocktrans %}

{% trans "4. 처리의 법적 근거" %}

  • {% blocktrans %}계약 이행의 필요: 이용자 요청에 따른 서비스 제공.{% endblocktrans %}
  • {% blocktrans %}정당한 이익: 서비스 개선, 보안 강화, 사기 방지(이익과 권리의 균형을 고려).{% endblocktrans %}
  • {% blocktrans %}동의: 선택적 마케팅 수신 등(언제든지 철회 가능).{% endblocktrans %}
  • {% blocktrans %}법적 의무 준수: 세무/회계 기록 보존, 법 집행기관 요청 대응 등.{% endblocktrans %}

{% trans "5. 개인정보의 공유" %}

  • {% blocktrans %}수탁사/처리자: 인프라(호스팅, 스토리지), 결제 대행, 분석/모니터링, 고객지원 등 서비스 운영을 위한 파트너와 계약에 따라 처리됩니다.{% endblocktrans %}
  • {% blocktrans %}법적 요청: 법령 준수 또는 권리/안전을 보호하기 위해 필요한 경우.{% endblocktrans %}
  • {% blocktrans %}사업 이전: 합병·인수·자산 양도의 경우, 본 방침과 동등한 수준으로 보호되는 조건에서 이전될 수 있습니다.{% endblocktrans %}

{% blocktrans %}당사는 법이 허용하는 범위에서, 이용자의 동의 없이 개인을 판매하지 않으며, 민감정보는 수집하지 않습니다.{% endblocktrans %}

{% trans "6. 국외 이전" %}

{% blocktrans %}서비스 제공 특성상 데이터가 거주 국가 밖의 서버로 이전·저장·처리될 수 있습니다. 이 경우 적용 법령을 준수하고, 적정성 결정, 표준계약조항(SCC) 또는 이에 준하는 적절한 보호조치를 적용합니다.{% endblocktrans %}

{% trans "7. 보관 기간" %}

  • {% blocktrans %}서비스 제공 및 법적 의무 이행에 필요한 기간 동안 보관합니다.{% endblocktrans %}
  • {% blocktrans %}구독 종료 후 30일 경과 시 업로드 원본과 결과물은 영구 삭제됩니다(관련 법령에 따라 예외가 있을 수 있음).{% endblocktrans %}
  • {% blocktrans %}회계/거래 기록 등은 법정 보존기간을 따릅니다.{% endblocktrans %}

{% trans "8. 보안" %}

{% blocktrans %}당사는 암호화, 접근통제, 모니터링 등 합리적인 기술적·관리적 조치를 적용합니다. 다만 인터넷을 통한 전송·저장은 100% 안전을 보장할 수 없으며, 최선의 보호를 위해 지속적으로 개선합니다.{% endblocktrans %}

{% trans "9. 이용자 권리" %}

  • {% blocktrans %}열람/사본 제공, 정정/삭제, 처리 제한, 이의 제기, 데이터 이동권을 행사할 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}동의를 기반으로 처리되는 경우, 언제든지 동의를 철회할 수 있습니다(철회 전 처리의 적법성은 영향을 받지 않음).{% endblocktrans %}
  • {% blocktrans %}법이 허용하는 범위에서, 부당한 반복·과도한 요청에는 합리적인 비용을 청구하거나 요청을 제한할 수 있습니다.{% endblocktrans %}
  • {% blocktrans %}관할 기관에 불복/민원을 제기할 권리가 있습니다.{% endblocktrans %}

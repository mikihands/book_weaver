지금 바로 시작하기

PDF를 드래그&드롭하거나 파일 선택

무료 사용자는 문서당 최대 30페이지까지 번역할 수 있습니다.

A

레이아웃 보존

표·도표·각주 등 구조를 최대한 유지한 HTML/PDF 생성.

B

빠른 시작

업로드 즉시 처리. 완료된 페이지부터 차례로 열람.

C

편집 지원

문단 재번역(Pro/Ent 무료)과 간단 편집툴 제공.

이용 방법

1. 업로드

PDF 선택 → 언어/모델 선택 → 시작.

2. 실시간 확인

완료된 페이지부터 브라우저에서 바로 보기.

3. 편집/출판

사소한 오류는 편집툴로 보정하고, 유료는 PDF로 다운로드.
